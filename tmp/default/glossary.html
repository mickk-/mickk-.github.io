simple model
A model of a given concept ‘without surprises’ and defined according to certain conventions described in Simple model.
function
A function-like constant object or constant template. Described in Function.
range function
A function which produces a model of Range.
strict range
A Range which fulfils all complexity requirements.
lax range
A Range which fulfils all complexity requirements, but only in an amortised manner.
